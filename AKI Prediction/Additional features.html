We can create a more balanced dataset by using oversampling or undersampling techniques. In this case, we have an imbalanced dataset where the AKI class (class 1) has very few samples compared to the NON-AKI class (class 0).

Undersampling is the process of randomly deleting samples from the majority class in order to balance the dataset. Oversampling, on the other hand, entails adding new synthetic samples in the minority class to balance the dataset.

df = pd.read_csv(url)

Undersampling:¶

This can be achieved by using sample() function which randomly selects a subset of the majority class that is equal in size to the minority class.

In [2]:
# Count the number of samples in each class
counts = df["KDIGO"].value_counts()

# Get the size of the minority class
min_class_size = counts.min()
In [3]:
# Sample randomly from the majority class to get a subset of the same size as the minority class
maj_class = df[df["KDIGO"] == 0].sample(n=min_class_size, random_state=42)

# Combine the minority class with the subset of the majority class
min_class = df[df["KDIGO"] == 1]
balanced_df = pd.concat([maj_class, min_class])
In [4]:
# Shuffle the dataset to avoid any ordering biases
balanced_df = balanced_df.sample(frac=1, random_state=42)

Oversampling¶

An approach to oversampling would be to use the Synthetic Minority Over-sampling Technique (SMOTE) to create new synthetic samples for the minority class. SMOTE involves creating synthetic samples by interpolating between existing samples of the minority class. In order to achieve this we can use the SMOTE function.

In [14]:
from imblearn.over_sampling import SMOTE
In [15]:
# Separate the features from the labels
X = df.drop("KDIGO", axis=1)
y = df["KDIGO"]
In [16]:
# Apply SMOTE to create new synthetic samples for the minority class
sm = SMOTE(random_state=42)
X_resampled, y_resampled = sm.fit_resample(X, y)
In [17]:
# Combine the features and labels into a new dataframe
balanced_df = pd.concat([X_resampled, y_resampled], axis=1)

# Shuffle the dataset to avoid any ordering biases
balanced_df = balanced_df.sample(frac=1, random_state=42)
